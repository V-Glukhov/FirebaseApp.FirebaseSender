сервис принимает из шины сообщения от InternalApi, отправляет их на сервер Firebase
Если отправка успешна - архивирует сообщения в рэдис и добавляет в БД MS SQL данные для работы модуля статистики
при ошибке - генерирует exception, обработчик которого возвращает сообщение в шину для повторной обработки (UseMessageRetry в Masstransit для RabbitMq)
